# ScholarFlux

App to fix scholar life mess.

A school management app — starter folder and scaffolding for Flutter mobile using Riverpod and go_router.

## iOS Simulator

**Requirement:** Full **Xcode** (not just Command Line Tools). The iOS Simulator is only included with Xcode.

1. **Install Xcode** from the [App Store](https://apps.apple.com/app/xcode/id497799835). Open it once and accept the license.
2. **Run the setup script** (switches to Xcode, runs first-launch, installs CocoaPods):

   ```bash
   cd mobile
   chmod +x setup_ios_simulator.sh
   ./setup_ios_simulator.sh
   ```

3. **Launch the simulator and run the app:**

   ```bash
   open -a Simulator
   cd mobile && flutter run
   ```
