# Gainx

Gainx is a Flutter project for the Gainx app. This repository contains the Flutter app source in the `lib/`, platform folders (`android/`, `ios/`), and project configuration files.

## Getting started

1. Install Flutter: https://docs.flutter.dev/get-started/install
2. Run: `flutter pub get`
3. To run on iOS or Android: `flutter run`
