# Gainx

Gainx is a low-volume/SBL AI personal trainer built with Flutter. This repository contains the app source and platform projects required to build and run the mobile app.

## Getting started

Prerequisites:
- Install Flutter: https://docs.flutter.dev/get-started/install
- Install Xcode (macOS) to run iOS builds
- Install Android SDK/Android Studio to run Android builds

Setup:
1. Clone the repository
2. Run `flutter pub get` to fetch dependencies
3. Run the app on a device or simulator: `flutter run`

Project layout:
- lib/: Dart source code and UI
- android/: Android platform files
- ios/: iOS platform files
- functions/: helper scripts and functions

Contributing:
- Open issues and pull requests.
- Follow the existing code style and run `flutter analyze` before submitting PRs.
